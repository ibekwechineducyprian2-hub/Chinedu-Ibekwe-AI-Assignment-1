# Mama Put Online - Prompt Documentation

## Prompt 1: Initial Concept
**Prompt:** 
"Design a vibrant, modern landing page for 'Mama Put Online', a Nigerian takeaway restaurant in Port Harcourt. Use warm colors like green, yellow, red. Include hero section, menu cards for Banga, Egusi, Jollof, sticky header, and mobile responsive layout. Use only HTML + CSS + JS."

**Output used:** Basic HTML structure + color scheme

## Prompt 2: Improve Visuals  
**Prompt:** 
"Make the hero section more emotional. Add overlay on background image, bigger headline 'Taste of Home, Delivered Hot', and an orange-red CTA button that scales on hover. Keep it fast and lightweight."

**Output used:** Hero CSS + button animation

## Prompt 3: Menu Section
**Prompt:** 
"Add a responsive grid of 3 menu cards with images, food names, short descriptions, and prices in Naira. Use CSS Grid with auto-fit. Add hover lift effect on cards. Food: Banga Soup, Egusi, Jollof Rice."

**Output used:** Menu grid + card styles

## Prompt 4: Mobile Fix
**Prompt:** 
"The nav links break on mobile. Hide nav and keep logo + make hero text smaller below 768px. Use media query."

**Output used:** @media query for responsiveness

## Prompt 5: Nigerian Vibe
**Prompt:** 
"Add Nigerian cultural touches: use emoji 🍲 in logo, footer text 'Made with ❤️ and pepper soup', and warm earthy background #FFFDF5 instead of white."

**Output used:** Final polish + branding

## Vibe Engineering Notes
Vibe = emotion + brand feel. For Mama Put I targeted:
1. **Warmth**: Yellow/Green/Red = Nigerian flag colors
2. **Trust**: Clean layout, clear prices  
3. **Appetite**: High-quality food images + short descriptions
4. **Speed**: No frameworks, loads fast on 3G

Total iterations: 5. Each prompt built on previous output.
